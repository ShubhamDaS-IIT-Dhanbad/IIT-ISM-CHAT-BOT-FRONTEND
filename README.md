# IIT-ISM Chatbot Frontend

A modern and user-friendly frontend interface for the IIT-ISM Chatbot, designed for the Indian Institute of Technology (Indian School of Mines), Dhanbad. This project provides an intuitive and responsive UI to interact with the chatbot backend, enabling seamless communication for students, faculty, and staff.
## 🌐 Live Demo

Check out the deployed version on Vercel:  
👉 **[https://bckd-pi.vercel.app]([https://bckd-pi.vercel.app](https://iit-ism-chatbot-frontend.vercel.app))**

## 🚀 Features

- ✅ API-driven chatbot backend  
- ✅ Python-powered logic and responses  
- ✅ Lightweight and easily deployable on Vercel  
- ✅ Modular code structure for easy scalability
## 🚀 Features

- **Responsive UI**: Optimized for both desktop and mobile devices.  
- **Interactive Chat Interface**: Real-time interaction with the IIT-ISM chatbot backend.  
- **Modern Design**: Clean and intuitive user experience with a focus on accessibility.  
- **Scalable Architecture**: Modular codebase for easy maintenance and future enhancements.  

## 🛠️ Tech Stack

- **Frontend**: React, JavaScript, HTML, CSS  
- **Styling**: Tailwind CSS  
- **Deployment Platform**: Vercel

Follow these steps to set up the project locally:

```bash
# Clone the repository
git clone https://github.com/ShubhamDaS-IIT-Dhanbad/IIT-ISM-CHAT-BOT-FRONTEND.git
cd IIT-ISM-CHAT-BOT-FRONTEND

# Install dependencies
npm install

# Run the application
npm start

```
